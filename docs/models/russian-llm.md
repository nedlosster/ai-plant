# Российские open-source LLM

Платформа: Radeon 8060S (120 GiB GPU-доступной памяти), llama-server + Vulkan. Модели с фокусом на русский язык.

## Зачем российские модели

Qwen2.5 и Llama обучены преимущественно на английском/китайском. Российские модели -- fine-tune на русских данных, дающий:
- Естественную русскую речь (не "переводной" стиль)
- Знание российского контекста (география, культура, законодательство)
- Корректное использование падежей, склонений, ударений
- Лучшее понимание русскоязычных задач (суммаризация, перевод, документация)

## Saiga (IlyaGusev)

**Назначение**: instruction-tuned русские модели. Самый известный русский fine-tune проект. Регулярно обновляется на лучших базовых моделях.

**Автор**: Ilya Gusev (независимый исследователь). HuggingFace: `IlyaGusev`.

**Сильные стороны**:
- Fine-tune на качественных русских instruction-данных
- Актуальные базовые модели (Qwen2.5, Llama 3.1)
- GGUF-квантизации от автора
- Публичные бенчмарки на MERA
- Активная поддержка и обновления

**Слабые стороны**:
- Качество ограничено базовой моделью (fine-tune не может превысить потолок base model)
- Нет специализации (финтех, медицина, юриспруденция)

**Применение**:
- Русскоязычный chat-бот, ассистент
- Суммаризация русских текстов
- Перевод (русский <-> английский)
- Генерация русского текста (статьи, документация)
- Тестирование русскоязычных пайплайнов

| Версия | Базовая модель | Размер | VRAM (Q4) | Качество |
|--------|---------------|--------|-----------|----------|
| saiga_qwen2.5_72b | Qwen2.5-72B | 72B | ~42 GiB | максимальное |
| saiga_qwen2.5_32b | Qwen2.5-32B | 32B | ~20 GiB | высокое |
| saiga_llama3.1_8b | Llama-3.1-8B | 8B | ~5 GiB | хорошее |

Saiga на Qwen-базе лучше для русского, чем на Llama-базе (Qwen изначально обучен на большем объеме русских данных).

## T-lite / T-pro (Т-Банк)

**Назначение**: русскоязычные модели от Т-Банка (ex-Тинькофф). Обучены на финтех-данных и общих русских корпусах.

**HuggingFace**: `T-Bank`

**Сильные стороны**:
- Корпоративное качество (обучение на реальных бизнес-данных)
- T-pro (72B) -- одна из сильнейших русских моделей
- Хорошее знание российского контекста (законодательство, финансы)
- Apache 2.0 лицензия

**Слабые стороны**:
- Обновления реже, чем у Saiga
- Ограниченная экосистема (меньше community-поддержки)

**Применение**:
- Бизнес-ассистент (финтех, юридические документы)
- Русскоязычный chat с контекстом российских реалий
- Анализ документов на русском
- Суммаризация договоров, регламентов

| Модель | Базовая | Размер | VRAM (Q4) |
|--------|---------|--------|-----------|
| T-pro | Qwen2.5-72B | 72B | ~42 GiB |
| T-lite | Qwen2.5-7B/8B | 7-8B | ~5 GiB |

## Vikhr (Vikhr AI)

**Назначение**: компактные русские модели с расширенным русским токенизатором.

**HuggingFace**: `Vikhrmodels`

**Сильные стороны**:
- Расширенный русский токенизатор (эффективнее кодирует кириллицу -- больше текста в контексте)
- Компактные размеры (7-13B)
- Apache 2.0

**Слабые стороны**:
- Базовые модели не самые сильные (Mistral 7B, Llama 2)
- Уступает Saiga на аналогичных размерах

**Применение**:
- Быстрый русский ассистент на ограниченных ресурсах
- Задачи, где важна эффективность токенизации русского текста
- Исследование русских NLP-подходов

## ruGPT-3.5 (AI-Forever / Сбер)

**Назначение**: русская GPT-модель, обученная с нуля (не fine-tune). Decoder-only, 13B.

**HuggingFace**: `ai-forever/ruGPT-3.5-13B`

**Сильные стороны**:
- Обучена с нуля на русских данных (не адаптация западной модели)
- Хорошее знание русского контекста

**Слабые стороны**:
- 13B -- по современным меркам малый размер
- Нет instruction-tuning из коробки (нужен fine-tune)
- Устаревает по сравнению с Saiga/T-pro на Qwen-базе

**Применение**:
- Базовая модель для fine-tuning на специализированных русских данных
- Исследование русских language models

## FRED-T5 (AI-Forever / Сбер)

**Назначение**: encoder-decoder модели для NLU-задач (не генерация, а понимание).

**HuggingFace**: `ai-forever/FRED-T5-large`, `ai-forever/FRED-T5-xl`

| Размер | Параметры | Назначение |
|--------|-----------|-----------|
| base | 250M | Классификация, NER |
| large | 820M | Суммаризация, QA |
| XL | 1.7B | Максимальное качество NLU |

**Ограничения**: encoder-decoder архитектура (T5), не поддерживается llama.cpp. Для использования нужен PyTorch.

**Применение**:
- Классификация русских текстов
- NER (Named Entity Recognition) для русского
- Суммаризация (extractive)
- QA (question answering) по русским документам

## Что выбрать

### Для русского chat-бота / ассистента

**Лучший вариант**: Saiga (Qwen2.5-32B) -- ~20 GiB, отличный русский, Apache 2.0.

**Максимальное качество**: Saiga (Qwen2.5-72B) или T-pro -- ~42 GiB.

**Быстрый/компактный**: T-lite или Saiga (Llama3.1-8B) -- ~5 GiB.

### Для бизнес-задач (финтех, юриспруденция)

**T-pro** -- обучена на бизнес-данных, знает российский контекст.

### Для NLU (классификация, NER)

**FRED-T5** -- encoder-decoder, требует PyTorch.

### Saiga vs Qwen2.5 (без fine-tune)

| Критерий | Saiga (Qwen2.5-32B) | Qwen2.5-32B-Instruct |
|----------|---------------------|---------------------|
| Русская речь | более естественная | хорошая, но "переводная" |
| Российский контекст | лучше | базовый |
| Instruction-following | хороший | отличный |
| Английский | хуже (fine-tune на русском) | отличный |
| Кодинг | хуже | хороший |

Для чисто русских задач -- Saiga. Для мультиязычных (русский + английский + код) -- Qwen2.5-Instruct.

## Бенчмарки русского языка

### MERA (mera.a-ai.ru)

Набор из ~20+ задач: знания, логика, этика, код. Лидерборд российских и международных моделей.

### RuMMLU

Русская адаптация MMLU. 57 предметных областей.

### Russian SuperGLUE (russiansuperglue.com)

Задачи NLU: NLI, QA, coreference. Классический бенчмарк.
