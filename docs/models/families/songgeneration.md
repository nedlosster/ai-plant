# SongGeneration v2 (Tencent, 2025)

> 4B полные песни с вокалом высокого качества, research-only лицензия.

**Тип**: 4B
**Лицензия**: Research only
**Статус на сервере**: не скачана
**Направления**: [music](../music.md)

## Обзор

SongGeneration v2 -- 4B-модель от Tencent для генерации полных песен с вокалом. Высокое качество. **Research-only** -- запрещено коммерческое использование, для коммерции -- [ace-step](ace-step.md) или [yue](yue.md).
