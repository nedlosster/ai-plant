# Stable Diffusion 3.5 (Stability AI, 2024)

> Эволюция SD: SD3.5 Large/Medium для генерации изображений.

**Тип**: diffusion (8B / 2.6B)
**Лицензия**: Stability AI Community License
**Статус на сервере**: не скачана
**Направления**: [images](../images.md)

## Обзор

Stable Diffusion 3.5 от Stability AI -- эволюция SD. Two размера: Large (8B) для качества, Medium (2.6B) для скорости. GGUF-квантизации от city96. Альтернатива FLUX в open-source стеке.

## Варианты

| Вариант | Параметры | GGUF Q8 | GGUF Q4 | Hub |
|---------|-----------|---------|---------|-----|
| SD 3.5 Large | 8B | ~12 GiB | ~7 GiB | [city96/sd3.5-large-gguf](https://huggingface.co/city96/sd3.5-large-gguf) |
| SD 3.5 Medium | 2.6B | ~4 GiB | ~2.5 GiB | [city96/sd3.5-medium-gguf](https://huggingface.co/city96/sd3.5-medium-gguf) |

## Сильные кейсы

- **Меньший VRAM** чем у [flux](flux.md)
- **Зрелая экосистема SD** -- максимум LoRA, ControlNet, integrations
- **Multiple aspect ratios** из коробки
- **Медиум для слабых GPU** (~4 GiB Q8)

## Слабые стороны

- Качество уступает [flux](flux.md) на photo-realistic
- Stability CL -- ограничения для коммерции
- Хуже на длинных промптах (нет T5-XXL по умолчанию)

## Идеальные сценарии

- Когда нужен **меньший VRAM** чем FLUX
- Доступ к **большой LoRA-библиотеке SD**
- Прототипирование при быстрых итерациях
- Backwards-compatible с SD 2/SDXL workflows

## Загрузка

```bash
huggingface-cli download city96/sd3.5-large-gguf --include "*Q4_K*" --local-dir ComfyUI/models/diffusion_models/
```
