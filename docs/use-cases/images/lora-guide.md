# LoRA для генерации изображений

Поиск, использование и тренировка LoRA-адаптеров для diffusion-моделей.

## Что такое LoRA

LoRA (Low-Rank Adaptation) -- легковесные адаптеры, которые модифицируют поведение базовой модели без полного дообучения. Файл LoRA занимает 10-300 MiB (против 6-18 GiB для полной модели).

LoRA добавляют:
- Стили (фотореализм, аниме, конкретный художник)
- Концепты (конкретный персонаж, объект, текстура)
- Улучшения (детализация рук, лиц, текста)

## Где брать LoRA

### CivitAI (civitai.com)

Крупнейший источник LoRA для diffusion-моделей. Фильтры по базовой модели (FLUX, SDXL, SD 1.5), типу (стиль, персонаж, концепт), рейтингу.

При скачивании обращать внимание на:
- **Base Model** -- LoRA совместима только с указанной базовой моделью
- **Trigger words** -- ключевые слова для активации LoRA
- **Примеры** -- галерея результатов с промптами и параметрами
- **Рейтинг и количество скачиваний** -- показатель качества

### HuggingFace

Официальные и community LoRA. Поиск по тегам `lora`, `flux`, `sdxl`.

## Как выбрать LoRA

1. **Совместимость базовой модели.** FLUX LoRA не работает с SD 3.5 и наоборот. Проверять поле "Base Model" на CivitAI.
2. **Рейтинг.** Модели с высоким рейтингом и большим числом скачиваний -- надежнее.
3. **Примеры результатов.** Проверять галерею на CivitAI -- промпты и настройки указаны под каждым изображением.
4. **Размер файла.** LoRA от 10 до 300 MiB. Большие файлы (>150 MiB) -- обычно более выразительные, но не всегда лучше.
5. **Версия.** Выбирать последнюю версию, если у LoRA несколько релизов.

## Использование LoRA в ComfyUI

### Подключение

1. Скачать .safetensors файл LoRA в `ComfyUI/models/loras/`
2. Добавить ноду **Load LoRA** в workflow
3. Подключить входы model и clip от загрузчика модели
4. Подключить выходы model и clip к KSampler и CLIP Text Encode

```
Unet Loader --> Load LoRA --> KSampler
CLIP Loader --> Load LoRA --> CLIP Text Encode
```

### Параметры Load LoRA

| Параметр | Описание | Диапазон |
|----------|----------|----------|
| strength_model | Влияние LoRA на модель (генерация) | 0.0-2.0, рекомендуется 0.5-1.0 |
| strength_clip | Влияние LoRA на text encoder (промпт) | 0.0-2.0, рекомендуется 0.5-1.0 |

- strength 1.0 -- полное влияние LoRA
- strength 0.5-0.7 -- мягкое смешивание с базовой моделью
- strength >1.0 -- усиление эффекта (может вызвать артефакты)

Типичный подход: начать с strength_model=0.8, strength_clip=0.8, затем корректировать.

## Стекинг нескольких LoRA

Можно применить несколько LoRA последовательно:

```
Unet Loader --> Load LoRA (стиль) --> Load LoRA (персонаж) --> KSampler
CLIP Loader --> Load LoRA (стиль) --> Load LoRA (персонаж) --> CLIP Text Encode
```

Рекомендации:
- Не более 2-3 LoRA одновременно -- артефакты при стекинге растут
- Снижать strength каждой LoRA при стекинге (0.5-0.7 вместо 1.0)
- Стиль + концепт -- хорошая комбинация
- Два стиля -- конфликтуют чаще

## Trigger words

Многие LoRA требуют trigger word (ключевое слово) в промпте для активации.

Пример: LoRA для стиля "pixel art" может требовать trigger word `pixelart style`.

Где найти:
- На странице LoRA на CivitAI -- раздел "Trigger Words"
- В описании модели на HuggingFace
- В примерах промптов (обычно выделено жирным)

Без trigger word LoRA может не активироваться или работать нестабильно.

## Тренировка LoRA

При 96 GiB VRAM тренировка LoRA доступна для всех популярных моделей.

### Инструменты

| Инструмент | Описание |
|------------|----------|
| **kohya_ss** | Наиболее популярный. GUI и CLI. Поддержка SDXL, SD 1.5, FLUX |
| **SimpleTuner** | Упрощенный тренер для FLUX, SDXL. Хорошая поддержка AMD |

### Потребление VRAM при тренировке

| Модель | VRAM (fp16) | VRAM (fp8) |
|--------|-------------|------------|
| SD 1.5 | ~8 GiB | ~6 GiB |
| SDXL | ~16 GiB | ~10 GiB |
| FLUX.1 Dev | ~30 GiB | ~18 GiB |

96 GiB -- с запасом для любой модели. Можно тренировать в fp16 без оптимизаций по памяти.

### Общий процесс

1. Подготовка датасета: 10-50 изображений + текстовые описания (caption)
2. Настройка параметров: rank, learning rate, epochs
3. Тренировка: обычно 500-3000 steps
4. Проверка: генерация с разными промптами и strength

Подробнее о тренировке diffusion-моделей: [training/diffusion-finetuning.md](../../training/diffusion-finetuning.md).
