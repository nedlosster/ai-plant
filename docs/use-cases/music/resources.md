## Сравнение с альтернативами

| Модель | Тип | Вокал | Русский | Локально | Лицензия | Стоимость |
|--------|-----|-------|---------|----------|----------|-----------|
| **ACE-Step 1.5** | Open-source | да | да | да | MIT | бесплатно |
| Suno v5 | Облачный | да | да | нет | проприетарная | $10+/мес |
| Udio v1.5 | Облачный | да | да | нет | проприетарная | $10+/мес |
| MusicGen | Open-source | нет | -- | да | MIT | бесплатно |
| Stable Audio | Open-source | нет | -- | да | Stability AI | бесплатно |
| YuE | Open-source | да | нет | да | Apache 2.0 | бесплатно |
| Bark | Open-source | частично | да | да | MIT | бесплатно |

ACE-Step 1.5 -- единственная open-source модель, генерирующая полные песни с вокалом на уровне коммерческих сервисов.
