# ACE-Step 1.5 -- генерация музыки с вокалом

Open-source модель генерации полных песен (вокал + инструменты) из текстового описания. MIT-лицензия. Поддержка 50+ языков, включая русский.

## Почему ACE-Step

- Полная песня за <10 секунд на GPU (turbo-режим)
- 50+ языков вокала, русский в топ-10
- Менее 4 GiB VRAM (DiT-only), 8-24 GiB с LM
- Поддержка AMD ROCm
- LoRA-тренировка на собственных данных
- Режимы: генерация, cover, remix, repaint, extract stems
- Качество на уровне Suno v4--v4.5 по метрикам

## Архитектура

Двухстадийный pipeline:

1. **LM-планировщик** (Qwen-based, 0.6B/1.7B/4B) -- анализирует промпт, генерирует blueprint песни (BPM, тональность, структура, семантические коды)
2. **DiT** (Diffusion Transformer, ~2B) -- генерирует аудио из blueprint через flow matching

Выход: 48kHz стерео, WAV/MP3/FLAC.

## Конфигурации по VRAM

| VRAM | LM-модель | Качество |
|------|-----------|----------|
| <=6 GiB | нет (DiT-only) | базовое |
| 6-8 GiB | 0.6B | хорошее |
| 8-16 GiB | 1.7B | высокое |
| >=24 GiB | 4B | максимальное |

На данной платформе (96 GiB) -- 4B LM без ограничений.
